# Signal Decoder Workspace

This repository hosts **Signal Decoder**, an interactive web application for inspecting mysterious identifiers, short hashes, and other opaque tokens. The full Next.js implementation lives in `web/`.

## Project Layout

- `web/` – Next.js 14 application with the analyzer UI and supporting logic
- `README.md` – Repository overview (this file)

Refer to `web/README.md` for development scripts, deployment instructions, and implementation details.
